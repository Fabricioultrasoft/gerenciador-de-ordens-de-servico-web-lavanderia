•         Remover tapete

 

Observação: A opção remover tapete só é possível caso o tapete não esteja atrelado a nenhuma ordem de serviço, aberta ou fechada. Caso o tapete esteja atrelado a alguma ordem de serviço, é necessário usar a opção Ativar, na qual você pode desativar o tapete, para que o mesmo não esteja mais disponível para ordens de serviço futuras.

1. Selecione o tapete que deseja remover

 

 

 

2. Clique no botão Remover

 

               

 

 

3. Abrirá uma nova janela

 

Caso você queira excluir definitivamente o tapete selecionado clique no botão Sim caso contrário clique no botão Não.
