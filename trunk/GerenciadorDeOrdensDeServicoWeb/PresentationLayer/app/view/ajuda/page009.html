•         Adicionar novo usuário

 

1. Clique no botão Adicionar para adicionar o usuário

 

 

 

2. Abrirá uma nova janela

 

       

 

Digite o tipo nome do usuário desejado no campo Nome, digite uma senha para o novo usuário no campo Senha e confirme a senha no campo Confirmar Senha, clique no botão Adicionar Usuário.
